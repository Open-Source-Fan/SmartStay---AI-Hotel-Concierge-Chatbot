## SmartStay – AI Hotel Concierge Chatbot

### Overview
SmartStay is an AI-powered hotel concierge chatbotbuilt for the Gen AI 4 GenZ Hackathon to improve guest experience and reduce operational load using compressed knowledge (ScaleDown).

### System Design
- Frontend: HTML, CSS, JavaScript
- AI Layer: GenAI-based chatbot logic
- Knowledge Layer: Compressed hotel services & local guides

### User Flow
Guest → Dashboard → Concierge Chat → Service Request → Resolution → Insights

### Key Features
- AI Concierge Chat
- Guest Dashboard
- Insights & Analytics
- Multilingual support (simulated)
- Token-optimized responses

### Metrics
- 80% information compression
- 50% reduction in front-desk calls
- Improved response time & staff efficiency

### Limitations
- PMS integration is simulated
- Location services disabled intentionally

### Future Scope
- Live PMS & booking APIs
- Voice assistant
- Mobile app
